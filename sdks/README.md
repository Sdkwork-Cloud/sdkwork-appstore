# SDK Families

SDK generation workspace for `sdkwork-appstore`.

| Family | Authority | Authored input |
| --- | --- | --- |
| `sdkwork-appstore-sdk` | `sdkwork-appstore-open-api` | `apis/open-api/store/openapi.yaml` |
| `sdkwork-appstore-app-sdk` | `sdkwork-appstore-app-api` | `apis/app-api/store/openapi.yaml` |
| `sdkwork-appstore-backend-sdk` | `sdkwork-appstore-backend-api` | `apis/backend-api/store/openapi.yaml` |

Materialization and generation scripts will be added in the implementation phase under `tools/`.

See `docs/sdk/sdk-design.md` for integration rules.
